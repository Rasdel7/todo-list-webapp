# Smart To-Do List ✅

Full task manager with priorities, deadlines,
categories and completion tracking.

## Live Demo
[Click here](https://todo-list-webapp-kxtpvc7ya9zxfds8tai8mi.streamlit.app)

## Features
- Add tasks with priority, category and deadline
- Overdue task detection and alerts
- Today's task view
- Filter by priority and category
- Sort by deadline, priority or category
- Restore accidentally completed tasks
- Analytics — category pie and priority bar chart
- Download completed tasks as CSV

## Tools Used
- Python, Streamlit, Pandas, Matplotlib

## How to Run Locally
pip install streamlit pandas matplotlib
streamlit run app.py
